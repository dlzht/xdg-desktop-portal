### XDG Desktop Portal

XDG Desktop Portal allow Flatpak apps, and other desktop containment frameworks, to interact with the system in a secure and well defined way.

### Features

|                       | impl | doc | example | version    | description                                                                                                                         |
|-----------------------|-----|-----|---------|------------|-------------------------------------------------------------------------------------------------------------------------------------|
| Account               | ❌ | ❌  |    ❌     | 1          | query basic information about the user, like their name and avatar photo                                                            |
| Background            | ❌ | ❌ |    ❌     | 2          | let application run in the background or started automatically when the user logs in                                                |
| Camera                | ❌ | ❌ |    ❌     | -          | access camera devices, such as web cams                                                                                             |
| Clipboard             | ❌ | ❌ |    ❌     | -          | access system clipboard                                                                                                             |
| Document              | ❌ | ❌ |     ❌    | 5          | make files from the outside world available to sandboxed applications in a controlled way                                           |
| Launcher              | ❌ | ❌ |     ❌    | 1          | instal application launchers(.desktop files) which have an icon associated with them and which execute a command in the application |
| Email                 | ❌ | ❌ |     ❌    | 4          | request to send an email, optionally providing an address, subject, body and attachments                                            |
| File Chooser          | ❌ | ❌ |    ❌     | 4          | ask the user for access to files                                                                                                    |
| File Transfer         | ❌ | ❌ |    ❌     | 1          | transfer files between apps                                                                                                         |
| Game Mode             | ❌ | ❌ |    ❌     | 4          | access GameMode                                                                                                                     |
| Global Shortcuts      | ❌ | ❌ |    ❌     | 1          | create global shortcuts sessions, and register shortcuts                                                                            |
| Inhibit               | ❌ | ❌ |    ❌     | 3          | inhibit the user session from ending, suspending, idling or getting switched away                                                   |
| Input Capture         | ❌ | ❌ |     ❌    | 1          | capture input events from connected physical or logical devices                                                                     |
| Location              | ❌ | ❌ |     ❌    | 1          | query basic information about the location                                                                                          |
| Momory Monitor        | ❌ | ❌ |    ❌     | 1          | provides information about low system memory                                                                                        |
| Network Monitor       | ❌ | ❌ |    ❌     | 3          | provides network status information                                                                                                 |
| Notification          | ❌ | ❌ |    ❌     | 2          | send and withdraw notifications                                                                                                     |
| OpenURI               | ❌ | ❌ |    ❌     | 5          | open URIs (e.g. a http: link to the applications homepage) under the control of the user                                            |
| Power Profile Monitor | ❌ | ❌ |    ❌     | 1          | provides information about the user-selected system-wide power profile                                                              |
| Print                 | ❌ | ❌ |         | 3          | allows applications to print                                                                                                        |
| Proxy Resolver        | ❌ | ❌ |    ❌     | 1          | provides network proxy information                                                                                                  |
| Realtime              | ❌ | ❌ |    ❌     | 1          | set threads to realtime                                                                                                             |
| Remote Desktop        | ❌ | ❌ |    ❌     | 2          | create remote desktop sessions                                                                                                      |
| Request               | ❌ | ❌ |    ❌     | -          | shared request interface                                                                                                            |
| Screencast            | ❌ | ❌ |    ❌     | 5          | create screen cast sessions                                                                                                         |
| Screenshot            | ❌ | ❌ |    ❌     | 2          | request a screenshot                                                                                                                |
| Secret                | ❌ | ❌ |    ❌     | 1          | retrieve a per-application secret                                                                                                   |
| Session               | ❌ | ❌ |    ❌     | -          | shared session interface                                                                                                            |
| Settings              | ❌ | ❌ |    ❌     | 2          | provides read-only access to a small number of standardized host settings required for toolkits similar to XSettings                |
| Trash                 | ❌ | ❌ |    ❌     | 1          | send files to the trashcan                                                                                                          |
| Usb                   | ❌ | ❌ |    ❌     | 1          | monitor and request access to connected USB devices                                                                                 |
| Wallpaper             | ❌ | ❌ |    ❌     | 1          | set the user’s desktop background picture                                                                                           |
